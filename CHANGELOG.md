# Changelog

All notable changes to **Lap Code** are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-02

First public release on the VS Code Marketplace.

### Added
- Live countdown timer in the Activity Bar, with overtime shown in red and as a negative duration.
- Per-task time budget — set a planned duration before starting a problem.
- Session history with planned vs. actual time for every completed task.
- LeetCode and NeetCode source presets, with complexity and source badges.
- One-click open of a task's source file from the history list.
- Favourites, delete, and restart actions on history entries.
- Import / export of session history.
- Idle detection that auto-pauses the timer when you stop coding.
- Task completion animations and refined sidebar transitions.
- Responsive layout that adapts to sidebar resize.
